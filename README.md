### Prerequisites
- Python 3.9+
- OpenAI API Key

### Installation
1. Clone the repository:


2. Enter your OpenAI API Key in the sidebar.
3. Upload your Resume (PDF) and paste the Job Description.
4. Click **Boost My Career**!
